Detailed Description

The class is an abstract class. It represents abstract optimization method and should be the parent class for class which represents specific method.

The class represents Random Search (Simple) method of optimization. Next vector is being searched in small neighbourhood of current vector (with probability p) or in the whole area (with probability 1 - p). Delta determines the smallness of neighbourhood and decreases if search in neighbourhood were succesfull.

The class represents Polak-Ribiere Conjugate Gradient method of optimization.
